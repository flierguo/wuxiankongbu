
# 游戏数值调整指南 (Game Balance Adjustment Guide)

本文档详细说明如何通过三种方式调整游戏内的**生物属性**与**掉落装备属性**。

## 核心文件概览
- **全局难度 & 地图**: `_核心部分/世界配置.ts`
- **生物属性细节**: `_核心部分/_生物/生物属性.ts`
- **装备掉落细节**: `_核心部分/_装备/装备掉落.ts`

---

## 方式 1: 同时调整 (Global Adjustment)
**适用场景**: 只有当且仅当你想改变**整个游戏的难度梯度**（例如让[困难]变得更难，同时让该难度的装备也更强）时使用。

### 操作位置: `世界配置.ts`
修改 `难度等级` 常量。

```typescript
// 文件: _核心部分/世界配置.ts
export const 难度等级 = {
    简单: { 名称: '简单', 倍数: 1, ... },
    普通: { 名称: '普通', 倍数: 2, ... }, // 若改为 3，则该难度的怪物更强，掉落装备也更强
    // ...
}
```
**影响**:
- **怪物**: 血量、攻击、防御会直接乘以新的倍数。
- **装备**: 掉落装备的**基础属性**也会乘以新的倍数。
- **结果**: 战斗体验维持平衡（怪物变强了，玩家拿到的装备也变强了），但数值膨胀速度变快。

---

## 方式 2: 仅调整生物 (Adjust Monsters Only)
**适用场景**: 觉得怪物太肉打不动，或者怪物伤害太高秒人，但不想改变装备强度。

### 操作位置: `生物属性.ts`
修改 `数值平衡配置` 常量。

```typescript
// 文件: _核心部分/_生物/生物属性.ts
const 数值平衡配置 = {
    血量额外倍数: 10,       // 调大->怪更肉；调小->怪更脆
    攻击倍数最小: 2,        // \
    攻击倍数最大: 10,       // -> 调整怪物攻击力波动范围
    防御倍数最小: 1,        // \
    防御倍数最大: 3,        // -> 调大->怪更硬(难破防)；调小->更容易破防
    // ...
}
```
**进阶调整**: 如果只想调整 BOSS 强度，不影响小怪，可以修改 `TAG生物倍数`：
```typescript
export const TAG生物倍数 = {
    1: 1,      // 小怪
    5: 8,      // BOSS (调大这个值，BOSS会特异化增强)
    // ...
}
```

## 方式 3: 仅调整装备 (Adjust Equipment Only)
**适用场景**: 觉得爆率太低、惊喜感不足，或者装备属性太差（刮痧）。

### 操作位置: `装备掉落.ts`
修改 `数值平衡配置` 常量。

```typescript
// 文件: _核心部分/_装备/装备掉落.ts
const 数值平衡配置 = {
    基础属性倍数最小: 1.5,  // \
    基础属性倍数最大: 4.5,  // -> 调整普通掉落的属性波动。调大=整体装备变强
    
    惊喜掉落几率: 150,      // 调小(如100)=更容易出惊喜；调大=惊喜更稀有
    惊喜倍数最小: 2.5,      // \
    惊喜倍数最大: 4.5,      // -> 调整惊喜装备的强度。调大=惊喜感更强，更容易越级挑战
}
```

## 总结速查表

| 调整目标 | 修改文件 | 修改变量 | 预期效果 |
| :--- | :--- | :--- | :--- |
| **整体节奏/数值膨胀** | `世界配置.ts` | `难度等级.倍数` | 怪强装备强，数值变大 |
| **怪物血量/耐打度** | `生物属性.ts` | `血量额外倍数` | 怪变肉/脆，不影响伤害 |
| **怪物攻击/秒杀能力** | `生物属性.ts` | `攻击倍数最大/小` | 怪伤害变高/低 |
| **卡关/打不动怪** | `生物属性.ts` | `防御倍数最大` | **调小**此值可缓解刮痧 |
| **装备整体强度** | `装备掉落.ts` | `基础属性倍数` | 所有掉落装备变强 |
| **追梦/跳级体验** | `装备掉落.ts` | `惊喜倍数/几率` | 偶尔掉落的神器更强 |
