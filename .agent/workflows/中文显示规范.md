## 中文显示规范

在编写代码时，请遵循以下规则：

1. **变量命名**：使用中文命名，例如 `玩家名称`、`套装配置`、`目标玩家`
2. **函数命名**：使用中文命名，例如 `装备属性统计()`、`清空背包()`
3. **注释说明**：所有注释使用中文编写
4. **错误提示**：用户可见的消息使用中文，例如 `Player.SendMessage('参数格式错误')`
5. **日志输出**：调试信息使用中文便于理解

### 示例

```typescript
// 正确示范
const 玩家等级 = Player.GetLevel();
const 装备列表: TUserItem[] = [];

function 计算伤害(攻击力: number, 防御力: number): number {
    return 攻击力 - 防御力;
}

// 避免使用
const playerLevel = Player.GetLevel();
const equipmentList: TUserItem[] = [];
```
