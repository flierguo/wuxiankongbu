## 查阅说明书规范

在编写脚本代码时，如果遇到以下情况，**必须优先查阅 `91m2说明书` 文件夹**：

1. **不确定某个API的用法**
2. **不清楚某个功能如何实现**
3. **需要参考示例代码**
4. **遇到游戏引擎特有的方法或属性**

### 说明书位置

```
h:\10职业\无限恐怖\91m2说明书\
```

### 查阅步骤

1. 首先使用 `list_dir` 查看说明书目录结构
2. 根据需要的功能找到对应的文档
3. 使用 `view_file` 阅读相关示例和说明
4. 参考示例编写代码

### 常见查阅场景

- 玩家对象 (TPlayObject) 的方法和属性
- NPC对象 (TNormNpc) 的方法和属性
- 物品对象 (TUserItem) 的操作方法
- GameLib 全局函数的使用
- 地图、怪物、技能等系统的API
