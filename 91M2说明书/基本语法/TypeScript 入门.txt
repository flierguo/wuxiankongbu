2、变量的基本类型
91引擎中会用到以下一些基础类型的变量 （其他变量类型参考官方文档）

数据类型	关键字	描述	实例
数字类型	number	双精度 64 位浮点值。它可以用来表示整数和分数	let i:number = 10
字符串类型	string	使用单引号（’）或双引号（”）来表示字符串类型	let name:string = “GM”
布尔类型	boolean	表示逻辑值：true 和 false。	let flag:boolen = true
数组类型	无	声明变量为数组。	let arr: number[] = [1, 2]
null	null	表示对象值缺失	let name:string = null
undefined	undefined	用于初始化变量为一个未定义的值	let name:string = undefined
any	any	代表任意类型	let i:any = 10
注：
如果不知道该设定什么类型 可以用 any, 但是不推荐这样做。

3、常量
如果一个值是确定好的，且程序执行过程中不会去改变它，那我们可以将其设置为常量

4、常量的声明

实例
//定义一个常量PI

//下面这行代码会报错，因为常量无法被修改
PI = 3.14

逻辑运算符
逻辑运算符用于测定变量或值之间的逻辑

运算符	描述	实例
&&	and	x < 10 && y > 1
||	or	x < 10 || y > 1
！	not	!(x==y)
实例
let avg: number = 20;
let percentage: number = 90;

console.log('avg 值为: ' + avg + ' ,percentage 值为: ' + percentage);

let res: boolean = avg > 50 && percentage > 80;
console.log('(avg>50)&&(percentage>80): ', res);

let res: boolean = avg > 50 || percentage > 80;
console.log('(avg>50)||(percentage>80): ', res);

let res: boolean = !(avg > 50 && percentage > 80);
console.log('!((avg>50)&&(percentage>80)): ', res);

1、条件语句
if 语句
只有当指定条件为 true 时，使用该语句来执行代码

实例
let num: number = 5;
if (num > 0) {
  console.log('数字是正数');
}
if…else 语句
当条件为 true 时执行代码，当条件为 false 时执行其他代码

实例
let num: number = 12;
if (num % 2 == 0) {
  console.log('偶数');
} else {
  console.log('奇数');
}
switch…case 语句
使用该语句来选择多个代码块之一来执行

实例
let grade: string = 'A';
switch (grade) {
  case 'A': {
    console.log('优');
    break;
  }
  case 'B': {
    console.log('良');
    break;
  }
  case 'C': {
    console.log('及格');
    break;
  }
  case 'D': {
    console.log('不及格');
    break;
  }
  default: {
    console.log('非法输入');
    break;
  }
}
2、循环语句
for 循环
实例
//循环10次
for (let i = 0; i < 10; i++) {
  console.log(i);
}
while 循环
//循环10此
let num = 1;
while (num < 10) {
  console.log(i);

3、结束语句
break 语句
break 语句有以下两种用法：

当 break 语句出现在一个循环内时，循环会立即终止，且程序流将继续执行紧接着循环的下一条语句。
它可用于终止 switch 语句中的一个 case。
continue 语句
continue 语句有点像 break 语句。但它不是强制终止，continue 会跳过当前循环中的代码，强迫开始下一次循环。

五、函数
函数是一组一起执行一个任务的语句。
可以把代码划分到不同的函数中。如何划分代码到不同的函数中是由自己来决定的.
但在逻辑上，划分通常是根据每个函数执行一个特定的任务来进行的。

1、函数定义
函数就是包裹在花括号中的代码块，前面使用了关键词 function：

2、语法格式
//定义函数
function add() {
  // 执行代码
  console.log("在此调用函数") 
}

//调用函数
add()
3、带参函数和返回值
function add(a: number, b: number): number {
  let c = a + b;
  return c;
}

let d = add(1, 2);
console.log('运行之后的值为:' + d);
